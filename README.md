#### WORDY
Solve English Word Games Easily

- installation
```bash
npm install @mohsenshafiei/wordy
```
- to use wordy:
```bash
wordy <alphabets> <minimum-length>
```
- example
```bash
wordy abcd 2
```
- result will be:
```bash
cab
abc
dab
bad
cad
dba
// wordy return all meaningful vocabularies
```
